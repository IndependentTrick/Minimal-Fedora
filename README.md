# Goal

Minimal Fedora Install. Tested for only Fedora 37.

# Installation

Use Fedora Everything.
Choose Minimal Install and Common NetworkManager Submodules

# Install DE of Choice

```
Usage: sudo bash install.sh [OPTIONS...]
    
       -b, --budgie, budgie
      Installs BUDGIE
       
       -g, --gnome, gnome
      Installs GNOME
      
       -p, --plasma, plasma
      Installs KDE PLASMA
      
```
